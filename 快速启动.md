## 🔧 手动启动服务

如果服务停止了，可以手动启动：

### 方法1: 使用批处理文件（推荐）
双击以下文件即可启动：

- `启动后端.bat` - 启动后端服务器
- `启动前端.bat` - 启动前端服务器

### 方法2: 命令行启动

**启动后端：**
```bash
cd c:/Users/DELL/CodeBuddy/route1/backend
node src/server.js
```

**启动前端（新终端）：**
```bash
cd c:/Users/DELL/CodeBuddy/route1/frontend
npm run dev
```

## ⚠️ 常见问题

### 1. 端口被占用
如果提示端口被占用，修改 `.env` 文件中的 `SERVER_PORT` 端口号。

### 2. 数据库连接失败
确保 MySQL 服务正在运行：
- 打开"服务"，找到"MySQL80"或类似名称
- 确保状态为"正在运行"

### 3. 前端无法连接后端
检查 `frontend/vite.config.ts` 中的代理配置，确保端口与 `.env` 中的 `SERVER_PORT` 一致。
