# 快速部署指南 - GitHub + Vercel

## 🎯 部署目标
- 单域名访问前后端
- 自动持续部署
- 全球 CDN 加速

---

## 📝 部署步骤（5分钟完成）

### 第一步：创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名：`campus-learning-platform`
3. 点击 `Create repository`

### 第二步：推送代码
```bash
# 运行项目中的 git-push.bat
git-push.bat
```

### 第三步：连接 Vercel
1. 访问 https://vercel.com/new
2. 用 GitHub 登录
3. 选择刚创建的仓库
4. 点击 `Import`

### 第四步：配置项目
在 Vercel 中配置：

**Build & Development Settings:**
- Framework Preset: `Other`
- Root Directory: `./`
- Build Command: `cd frontend && npm run build`
- Output Directory: `frontend/dist`

**Environment Variables**（必须）:
```
DB_HOST=your-db-host
DB_PORT=3306
DB_USER=your-db-user
DB_PASSWORD=your-db-password
DB_NAME=campus_learning_platform
JWT_SECRET=your-secret-key
NODE_ENV=production
```

### 第五步：部署
点击 `Deploy` 按钮，等待 1-3 分钟。

---

## 🌐 访问地址

部署成功后：

- **API**: `https://your-project.vercel.app/api/health`
