
## ⚠️ 注意事项

1. **如果页面显示"后端连接失败"**
   - 确保后端服务正在运行
   - 检查端口3005是否被占用
   - 查看后端终端窗口的错误信息

2. **如果页面空白**
   - 打开浏览器控制台（F12）查看错误
   - 确保前端服务正在运行
   - 尝试刷新页面（Ctrl+F5）

3. **登录问题**
   - 确认使用正确的测试账号
   - 检查后端终端是否有错误日志
   - 确保数据库连接正常

## 🔧 故障排除

### 端口被占用
如果提示端口被占用，修改以下文件：
- `backend/simple-server.js` 中的 `PORT` 常量
- `frontend/vite.config.ts` 中的 `proxy.target` 配置

### 数据库连接失败
确保MySQL服务正在运行：
1. 打开"服务"（services.msc）
2. 找到"MySQL"或"MySQL80"
3. 确保状态为"正在运行"

### 重置数据库
如需重新初始化数据库：
```bash
cd c:/Users/DELL/CodeBuddy/route1/backend
node src/database/init2.js
```
