## 🔍 故障排查

### 1. 后端启动失败

**错误：`EADDRINUSE: address already in use`**

解决方法：
```bash
# 查找占用端口的进程
powershell -Command "Get-NetTCPConnection -LocalPort 3006 -ErrorAction SilentlyContinue"

# 停止占用端口的进程
powershell -Command "Get-Process node | Stop-Process -Force"

# 重新启动后端
cd c:/Users/DELL/CodeBuddy/route1/backend
node simple-server.js
```

### 2. 数据库连接失败

**错误：`Access denied for user 'root'@'localhost'**

解决方法：
1. 确认MySQL正在运行
2. 确认MySQL密码是 `root`
3. 检查 `c:/Users/DELL/CodeBuddy/route1/.env` 文件：
   ```
   DB_PASSWORD=root
   ```

### 3. 登录仍然失败

检查步骤：

1. **确认后端正在运行**
   - 看后端终端窗口是否有错误
   - 访问 http://localhost:3006/api/health 应该返回 `{"status":"ok"}`

2. **确认前端连接到正确的端口**
   - 检查 `frontend/vite.config.ts` 中的 `proxy.target` 应该是 `http://localhost:3006`

3. **查看浏览器控制台错误**
   - 按F12打开开发者工具
   - 查看Console标签的错误信息
   - 查看Network标签的请求详情

### 4. 测试后端API

运行测试脚本：
```bash
cd c:/Users/DELL/CodeBuddy/route1/backend
node test-login.js
```
