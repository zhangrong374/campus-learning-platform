## 🔍 故障排查

### 问题：点击功能显示"接口不存在"

**解决**：
1. 确保使用 `完整启动.bat` 启动
2. 检查后端终端是否显示"后端服务器已启动"
3. 访问 http://localhost:5173/api-test.html 测试API

### 问题：登录或注册失败

**解决**：
1. 确认使用正确的测试账号
2. 检查后端终端是否有错误
3. 清除浏览器缓存（Ctrl+Shift+Delete）

### 问题：端口被占用

**解决**：
```bash
# 停止所有node进程
taskkill /F /IM node.exe

# 重新运行启动脚本
完整启动.bat
```

### 问题：数据库连接失败

**解决**：
1. 确认MySQL正在运行
2. 确认密码是 `root`
3. 手动运行：
   ```bash
   cd backend
   node init-users.js
   node fix-tables.js
   node create-missing-tables.js
   node init-data.js
   ```
